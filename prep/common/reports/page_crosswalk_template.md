# Page Crosswalk Template

## Purpose

Use this template to record printed-page and PDF-page mappings for the highest-
value source references used in prep work.

The point is simple:

- printed page is the primary human-facing citation
- PDF page is the extraction-tool and viewer reference

Recording both prevents provenance ambiguity later.

## Notes

- Do not guess printed-page or PDF-page values.
- If the document has appendix page labels such as `A-4` or `C-5`, use those
  in the `Printed page` column.
- If a document has no printed page labels, leave the printed-page entry blank
  and explain in `Notes`.
- Keep this document focused on high-value references rather than trying to
  crosswalk every page in the report.
